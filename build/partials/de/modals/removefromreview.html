Sind Sie sich sicher?

Indem Sie auf Entfernen drücken, wird die Police aus der Ansicht gelöscht. Derzeit wird die Police kostenfrei für Sie digitalisiert. Sofern Ihnen eine solche Versicherung von uns empfohlen wurde, wird eine entsprechende Empfehlung erscheinen bis Sie eine Police hochladen oder ein Angebot einholen.
